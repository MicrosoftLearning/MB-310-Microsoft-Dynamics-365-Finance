## Exercise 6: Process invoice and settle it against a payment

2. Select **New**.

3. In the **Customer account** field, enter or select **US-013**.

17. Set **Amount** to '**1,912.50'**.

18. Select **Save in journal** in the action pane. 

19. Close the page.

20. Select **Lines**.

22. Select **Validate**.

23. Select **Post**.
